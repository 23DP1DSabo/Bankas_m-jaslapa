Bankas sistēma

Iepazīsties kā strādāt ar banku bāzes funkcionalitāti

Par projektu

Šī bankas sistēmas programma ir izstrādāta Java valodā kā konsoles lietotne ar dažādu bibliotēku un frameworku palīdzību. Tās mērķis ir attēlot pamata bankas funkcionalitāti izglītojošiem nolūkiem. Tajā var sataisīt lietotāju, un lietotājs, savukārt, var izveidot kontu, pieslēgties esošam kontam, veikt iemaksas, izmaksas, pārskaitīt naudu, apskatīt konta atlikumu un citas darbības.

Galvenās funkcijas: a) Lietotāja reģistrācija b) Darbības ar digitālo naudu c) Konta atlikuma pārbaude d) Karšu (kredīta, debit un ISIC) izveidošana e) Nodokļu un kredītu kalkulatori

Šī sistēma nav paredzēta izmantot reāliem finansu nolūkiem bet gan kā mācību projekts, lai iepazīstinātu ar bankas bāzu funkcionalitāti un nodemonstrētu objektorientētās programmēšanas pamatus, failu apstrādi un lietotāja interfeisa izveidi konsolē.

Lejupielāde

(Totally not a scam!!!)

Spiediet uz pogas augstāk, lai lejupielādētu bankas sistēmau ZIP formātā. Pēc lejupielādes izvērsiet arhīvu un palaidiet programmu, izmantojot Java komandrindu vai piemērotu IDE rīku. Programma darbojas konsolē un neprasa instalēšanu. Pārliecinieties, ka jūsu datorā ir uzstādīta Java (versija 8 vai jaunāka).
